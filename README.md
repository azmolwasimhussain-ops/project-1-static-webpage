# DecodeLabs | Project 1 — Static Webpage

A clean, professional static webpage built using **semantic HTML5** and **vanilla CSS3** as part of the DecodeLabs curriculum.

---

## 📌 Project Overview

This project demonstrates core web development fundamentals including structured HTML markup, responsive CSS layouts using **Grid** and **Flexbox**, and a sticky navigation bar — all without any JavaScript or external frameworks.

---

## 🗂️ File Structure

```
project-1-static-webpage/
├── index.html      # Main HTML document
├── style.css       # External stylesheet
└── README.md       # Project documentation
```

---

## ✨ Features

- ✅ Semantic HTML5 elements (`header`, `main`, `section`, `article`, `footer`)
- ✅ Sticky navigation bar with anchor-based scroll links
- ✅ Hero section with a call-to-action button
- ✅ Responsive service cards using CSS Grid
- ✅ Flexbox-based layout utilities
- ✅ Clean typography and consistent spacing
- ✅ No JavaScript — pure HTML & CSS

---

## 🧭 Sections

| Section   | Description                                      |

| About     | Overview of DecodeLabs and its learning approach |

## 🚀 Getting Started

No build tools required. Simply open the file in your browser:

```bash
# Clone the repository
git clone https://github.com/azmolwasimhussain-ops/project-1-static-webpage.git

# Navigate into the project
cd project-1-static-webpage

# Open in browser
open index.html
```
